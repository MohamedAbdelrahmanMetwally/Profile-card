# Profile Card (business_card)

A simple, responsive profile card built with Flutter — a clean starter project that shows a personal avatar, name, title, contact actions, and social links.

---

## Features

* Minimal and clean design
* Responsive layout for mobile & web
* Customizable avatar, colors, fonts, and links
* Ready-to-run Flutter project structure (Android / iOS / Web / Desktop)

## Getting Started

### Prerequisites

* Flutter SDK (stable channel) — [https://flutter.dev](https://flutter.dev)
* Android Studio / Xcode (optional for emulators)
* A device or emulator

### Run Locally

```bash
# clone the repo
git clone https://github.com/MohamedAbdelrahmanMetwally/Profile-card.git
cd Profile-card

# get dependencies
flutter pub get

# run on an available device (phone, emulator, or web)
flutter run
```

For web: `flutter run -d chrome`

---

## Customization

* **Avatar:** Replace the image in `images/` and update its path inside the Flutter code.
* **Colors & Theme:** Edit theme values or color variables inside `lib/`.
* **Links & Contact:** Update icons and contact links in the main UI file.
